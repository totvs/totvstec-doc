## Adicionar outro projeto

1. Crie o repositório com `docs/conceito/`, `docs/exemplos/` e `docs/sidebar.json`
2. Adicione uma entrada em `repos.config.json` no [totvstec-doc](https://github.com/totvs/totvstec-doc)
3. Rode `npm run ecosystem:export` e `npm run validate:repos`

Detalhes: [Arquitetura modular](/docs/ARQUITETURA).
