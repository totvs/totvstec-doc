Fontes institucionais TOTVS
===========================

Arquivos oficiais (proprietários — uso restrito à TOTVS):
https://marca.totvs.com/downloads/ → Fonte-TOTVS.zip

Cópias usadas pelo build:
- src/fonts/totvs/ (empacotadas pelo webpack)
- static/fonts/totvs/ (espelho para referência)

Pesos carregados: Regular (400), SemiBold (500/600), Bold (700).

Dúvidas: marca@totvs.com.br
